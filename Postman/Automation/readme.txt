# 🧪 API Testing con Postman

¡Bienvenidos a mi carpeta de proyectos de automatización de pruebas sobre APIs!  
Aquí encontrarás distintos trabajos que realicé utilizando **Postman** como herramienta principal para realizar pruebas funcionales, de validación y automatización sobre endpoints REST.

## 🚀 ¿Qué vas a encontrar?

- Colecciones de pruebas en formato `.json` exportadas desde Postman
- Documentación clara de cada API testeada
- Casos de prueba (manuales y automatizados) detallando:
  - Método (GET, POST, PUT, DELETE)
  - Headers y parámetros utilizados
  - Body de ejemplo (cuando corresponda)
  - Validaciones esperadas
- Uso de variables de entorno y pre-scripts
- Automatización de test suites con **Collection Runner** y reportes

## 🛠️ Tecnologías y herramientas utilizadas

- [Postman](https://www.postman.com/) – Herramienta de testing de APIs
- [Newman](https://www.npmjs.com/package/newman) (en algunos proyectos) – Para ejecutar tests desde la terminal y generar reportes
- JSON / REST
- Prácticas de QA Testing aplicadas al mundo de APIs
