# 📬 Proyectos de API Testing con Postman

En esta carpeta se encuentran varios proyectos dedicados a la prueba y documentación de diversas APIs utilizando **Postman**. Cada proyecto se enfoca en el testing de una o más APIs específicas.

## 🛠️ Herramientas Utilizadas

# Postman:** Plataforma principal para la creación y organización de colecciones de API, definición de entornos y ejecución de pruebas en cada proyecto.

## 📂 Contenido

Esta carpeta contiene los archivos y recursos relacionados con mis proyectos de API testing utilizando Postman. Cada proyecto puede incluir colecciones de Postman (en formato JSON) y archivos de entorno para diferentes configuraciones de la API.

## 🎯 Pruebas Realizadas

Los tipos de pruebas realizadas variarán entre los diferentes proyectos. Esto puede incluir validación de respuestas, pruebas de métodos HTTP, verificación de códigos de estado, validación de esquemas, pruebas de autenticación, entre otros.
